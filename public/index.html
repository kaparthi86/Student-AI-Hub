Make this your start page?

Browsers don’t let websites change your home or startup page for you (by design). If you’d like Student AI Hub to open when you start the browser, you can set it yourself in a few seconds.

Copy this address:

  1. Chrome / Edge: Settings → On startup → Open a specific page → Add a new page → paste the address above. (Or visit chrome://settings/onStartup in Chrome.)
  2. Safari (Mac): Safari → Settings → General → Homepage — paste the address.
  3. iPhone / iPad: Share → Add to Home Screen for a quick icon (separate from the browser home page).

Using our Chrome extension? After you install it, new tabs can open this site automatically.
